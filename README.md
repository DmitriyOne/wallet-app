# Wallet App

Wallet and transaction management application. Built with React Router 7 using Feature Sliced Design architecture.

## Tech Stack

- React 19
- React Router 7
- TypeScript
- TailwindCSS
- Vite

## Getting Started

### Install Dependencies

```bash
npm install
```

### Development

```bash
npm run dev
```

Application will be available at `http://localhost:5173`

### Production Build

```bash
npm run build
```

### Run Production Build

```bash
npm start
```

### Type Checking

```bash
npm run typecheck
```

## Project Structure

Project uses Feature Sliced Design (FSD):

- `src/01_app/` — application initialization
- `src/02_pages/` — pages
- `src/03_widgets/` — large UI blocks
- `src/04_features/` — business logic
- `src/05_entities/` — business entities
- `src/06_shared/` — reusable modules
